# Tower Defense Game

A 3D tower defense game built with Babylon.js and TypeScript.

## Features

- 3D valley environment with two towers (player vs AI)
- Resource collection and management
- Modular tower building system
- Wave-based enemy attacks
- Servant command system
- Power level competition

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Technology Stack

- Babylon.js 6.0
- TypeScript
- Vite
